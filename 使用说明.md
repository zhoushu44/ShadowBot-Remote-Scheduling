# 远程调度查询 - 使用说明

## 功能介绍

本程序是一个基于 Flask 的 Webhook 服务，支持远程触发 BAT 文件执行、FRP 内网穿透、AI 屏幕分析和图片点击功能。

## 主要功能

### 1. Webhook 触发 BAT 执行
- 根据关键字触发同名的 BAT 文件执行
- 支持中文、英文、数字文件名

### 2. FRP 内网穿透
- 内置 frpc.exe（版本 0.65.0）
- 支持多配置管理（TCP/UDP）
- 无需手动选择 frpc 路径

### 3. AI 屏幕分析
- 截取当前屏幕
- 调用 AI API 分析屏幕内容
- 提取影刀运行状态信息

### 4. 图片点击
- 内置 4 个图片（start.png、Stop.png、Close.png、Pause.png）
- 通过 HTTP 接口点击屏幕上的图片
- 支持单次点击和组合操作

## 配置说明

### 配置文件
程序首次运行会自动生成 `config.json`，包含以下配置：

```json
{
  "bat_folder": "D:\\WebhookBats",
  "listen_port": 8888,
  "autostart": false,
  "frp": {
    "frpc_path": "",
    "configs": {}
  },
  "ai": {
    "api_url": "",
    "api_key": "",
    "model": "",
    "prompt": "请分析这张截图，描述当前屏幕上显示的内容和进度状态。"
  }
}
```

### 配置项说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| bat_folder | BAT 文件所在目录 | D:\WebhookBats |
| listen_port | Web 服务监听端口 | 8888 |
| autostart | 开机自启 | false |
| frp.frpc_path | frpc.exe 路径（已内置，留空即可） | "" |
| frp.configs | FRP 配置列表 | {} |
| ai.api_url | AI API 地址 | "" |
| ai.api_key | AI API 密钥 | "" |
| ai.model | AI 模型名称 | "" |
| ai.prompt | AI 分析提示词 | "请分析这张截图..." |

## HTTP 接口说明

### 基础接口

#### 1. Webhook 触发 BAT
```
POST /webhook
Content-Type: application/json

{
  "key": "文件名"
}
```

**curl 示例：**
```bash
curl -X POST "http://127.0.0.1:8888/webhook" \
  -H "Content-Type: application/json" \
  -d '{"key": "我的任务"}'
```

**响应：**
```json
{
  "code": 0,
  "msg": "文件名.bat 已触发"
}
```

**特殊 key：**
- `查询` - 触发 AI 分析，返回屏幕截图和分析结果

#### 2. 查询进度
```
POST /query_progress
```

**curl 示例：**
```bash
curl -X POST "http://127.0.0.1:8888/query_progress"
```

**响应：**
```json
{
  "code": 0,
  "msg": "查询成功",
  "screenshot": "base64编码的截图",
  "ai_analysis": "AI分析结果"
}
```

### 图片点击接口

#### 1. 点击开始按钮
```
POST /start
```

**curl 示例：**
```bash
curl -X POST "http://127.0.0.1:8888/start"
```

**响应：**
```json
{
  "code": 0,
  "msg": "start.png 点击成功",
  "coordinates": {"x": 100, "y": 200}
}
```

#### 2. 点击停止按钮
```
POST /stop
```

**curl 示例：**
```bash
curl -X POST "http://127.0.0.1:8888/stop"
```

#### 3. 点击关闭按钮
```
POST /close
```

**curl 示例：**
```bash
curl -X POST "http://127.0.0.1:8888/close"
```

#### 4. 点击暂停按钮
```
POST /pause
```

**curl 示例：**
```bash
curl -X POST "http://127.0.0.1:8888/pause"
```

#### 5. 停止并关闭（组合操作）
```
POST /stop_close
```

**curl 示例：**
```bash
curl -X POST "http://127.0.0.1:8888/stop_close"
```

**说明：** 先点击 Stop.png，等待 4 秒，再点击 Close.png

### 通用图片点击接口
```
POST /click_image
Content-Type: application/json

{
  "image_name": "图片名称"
}
```

**参数说明：**
- `image_name` - 图片文件名（支持带或不带 .png 后缀）

**响应：**
```json
{
  "code": 0,
  "msg": "图片名 点击成功",
  "coordinates": {"x": 100, "y": 200}
}
```

## 错误码说明

### 添加 FRP 配置
1. 打开程序，点击"FRP 设置"
2. 点击"添加配置"
3. 填写服务器信息：
   - frps 地址：FRP 服务器地址
   - 端口：FRP 服务器端口（默认 7000）
   - token：认证令牌
   - 链接方式：TCP 或 UDP
   - 名字：配置名称（自动生成随机名）
   - 本地地址：127.0.0.1
   - 本地端口：8888（webhook 服务端口）
   - 映射端口：服务器端开放端口
4. 点击"保存"

### 启动 FRP
1. 在 FRP 设置窗口选择配置
2. 点击"启动"
3. 查看日志确认连接状态

### FRP 使用
启动 FRP 后，可以通过以下地址访问：
```
http://frps地址:映射端口/webhook
```

## AI 配置

### 配置 AI
1. 打开程序，点击"AI 配置"
2. 填写 API 信息：
   - API URL：AI 服务地址
   - API Key：认证密钥
   - Model：模型名称（如：doubao-seed-2-0-pro-260215）
   - 提示词：分析提示词
3. 点击"保存配置"
4. 点击"测试连接"验证配置

### 使用 AI 查询
调用 `/query_progress` 接口，程序会：
1. 截取当前屏幕
2. 发送给 AI 分析
3. 返回分析结果

## 故障排查

### 问题 1：FRP 连接失败
**可能原因：**
- FRP 服务端未运行
- 服务器地址或端口错误
- Token 不匹配
- 防火墙阻止

**解决方法：**
1. 检查 FRP 服务端是否正常运行
2. 验证服务器地址和端口
3. 确认 Token 是否正确
4. 检查防火墙设置

### 问题 2：图片点击失败
**可能原因：**
- 屏幕上没有对应的图片
- 图片与屏幕显示不完全一致
- 置信度设置过高

**解决方法：**
1. 确保目标图片在屏幕上可见
2. 截取屏幕图片，确保与内置图片一致
3. 程序会自动降低置信度重试

### 问题 3：AI 分析失败
**可能原因：**
- 未配置 API Key 或 Model
- API 服务不可用
- 网络连接问题

**解决方法：**
1. 检查 AI 配置是否完整
2. 测试网络连接
3. 查看 AI 服务状态
